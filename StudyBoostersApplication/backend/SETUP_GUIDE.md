## ğŸ”‘ API Endpoints

### Auth
- `POST /api/auth/login` - Login
- `GET /api/auth/me` - Current user
- `POST /api/auth/promote-admin` - Become admin

### Files
- `GET /api/files` - List files
- `POST /api/files/upload` - Upload
- `POST /api/files/{id}/approve` - Approve (admin)
- `DELETE /api/files/{id}` - Delete (admin)
- `POST /api/files/{id}/download` - Download
